# Recursion

This project is part of [The Odin Project's Full Stack JavaScript path](https://www.theodinproject.com/paths/full-stack-javascript). It contains recursive implementations of the Fibonacci sequence and the merge sort algorithm.

## Installation

1. Clone the repository
2. Navigate to the project folder: `cd recursion`
3. Install dependencies: `npm install`
4. Run any file using: `node <file-name>`
